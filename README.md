Este repositorio contiene el ecosistema digital del **Centro Perfecta Alegría**, una institución dedicada al bienestar integral y acompañamiento biopsicosocial en Quito, Ecuador. El sitio ha sido desarrollado con un enfoque humano y profesional, optimizado para ofrecer una experiencia de usuario clara y accesible en cualquier dispositivo.

---

## 🚀 Sobre el Proyecto

El objetivo principal fue transformar la presencia digital del centro, migrando de un diseño estático a una plataforma optimizada, con una arquitectura de información que facilita a los pacientes y familias encontrar servicios de salud mental y psicopedagogía.

### Características Principales:
- **Diseño Responsivo:** Adaptabilidad total en móviles, tablets y escritorio.
- **Optimización SEO:** Implementación de meta-etiquetas estratégicas para mejorar el posicionamiento local.
- **Enfoque Clínico-Profesional:** Estructura pensada para transmitir confianza y calidez humana.
- **Integración de Contacto Directo:** Acceso rápido a WhatsApp para agendamiento de citas.

## 🛠️ Tecnologías Utilizadas

Como desarrollador enfocado en el rendimiento y la escalabilidad, utilicé las siguientes tecnologías:

- **HTML5:** Estructura semántica para accesibilidad.
- **CSS3 / SCSS:** Estilos modulares y variables para facilitar el mantenimiento de la identidad visual.
- **Vanilla JavaScript:** Interactividad ligera y rápida sin dependencias pesadas.
- **FontAwesome:** Iconografía profesional para una mejor guía visual.

## 📂 Estructura del Repositorio

- `index.html`: Página principal con el banner de impacto y visión general.
- `generic.html`: Estructura para la descripción detallada de servicios y tratamientos.
- `elements.html`: Catálogo de áreas de apoyo especializado.
- `assets/`: Directorio que contiene los estilos (SCSS/CSS), scripts y recursos multimedia.
